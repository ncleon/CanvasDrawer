Guide
1. To view the source and compile source, please using Maven & Eclipse (or similar Java IDE)
2. Immediate run the program can using command java -jar cs-canvas-0.0.1-SNAPSHOT.jar

Details:

-This project created using Maven and Eclipse.
-This project created using Acceptance Test Driven development practices. Therefore it is started from SimpleConsoleDrawTest
-SimpleConsoleDrawTest is to represent the user input test case, which using accept command and print the output. Which it will focus to test expected result printed after user input command
-CanvasDrawerTest represent unit test to ensure the core feature class CanvasDrawer will always return the expected result.
-Add additonal Command H, which to brief user how to use the program.
